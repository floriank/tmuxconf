# tmuxconf
My tmux configuration.

## Usage

:warning: [Prerequisite steps](https://github.com/tmux-plugins/tpm#installation)

```bash
$ git clone git@github.com:floriank/tmuxconf ~/code/tmuxconf
$ cd 
$ ln -s ~/code/tmuxconf/.tmux.conf
$ tmux
```
